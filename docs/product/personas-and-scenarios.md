# Personas and Core Scenarios

## Primary: Independent builder

A developer or designer working on self-directed projects. The main failure mode is activation and context switching, not lack of technical competence.

**Scenario:** Select “Implement login validation,” define “invalid inputs display errors and tests pass,” run Deep Formation for 50 minutes, record the next action.

## Secondary: Student

Works through reading, assignments or exam preparation. Needs small finishable outcomes and a visible work ritual.

## Secondary: Writer/researcher

Needs long uninterrupted intervals but may prefer Flow Mode or 50-minute sessions rather than a rigid 25-minute cycle.

## Excluded initial persona

Teams requiring shared project management, manager dashboards or employee surveillance. VIGIL is not a workforce monitoring product.
