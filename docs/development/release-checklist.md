# Release Checklist

- [ ] Target phase exit criteria pass.
- [ ] `npm run check` passes.
- [ ] Rust format, clippy and tests pass.
- [ ] Version synchronized (`npm run version:check`).
- [ ] Changelog updated.
- [ ] Data migration tested from previous released version.
- [ ] Clean install and uninstall tested.
- [ ] Active-session upgrade behavior tested.
- [ ] Tauri permissions reviewed.
- [ ] Assets regenerated and manifest current.
- [ ] Installer signature and hashes recorded when distribution begins.
