# Pending Asset Requests

Each request is one Markdown file named:

```text
AR-<sequence>-<asset-id>-<short-name>.md
```

A request must state exact dimensions, states, format, transparency, immutable details, reference IDs and the version blocked by the missing file.

Do not create a request merely to document a guess. The request is the point at which the implementation agent stops and asks the user.
