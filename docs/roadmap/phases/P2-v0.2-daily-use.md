# P2 — v0.2.0 Daily-use Alpha

## Objective

Make VIGIL usable as a daily personal focus tool for seven consecutive days.

## Features

- Campaign entity and mission attribution.
- Debrief: result, blocker and next action.
- Break state and Doctrine preferences.
- Weekly local summary.
- Tray behavior.
- Global shortcut.
- Native completion notification.
- Autostart preference.
- Companion side, scale, opacity and fullscreen preference.
- Export/import of local data.

## Exit criteria

- Seven-day dogfood run without data loss.
- Preferences recover correctly.
- Notification and shortcut permissions fail gracefully.
- Debrief adds less than 20 seconds median closure time.
