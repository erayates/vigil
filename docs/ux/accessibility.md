# Accessibility Requirements

- All actions available by keyboard.
- Visible focus indication with at least 3 px contrast boundary.
- Timer state announced without announcing every second.
- Companion meaning cannot rely on animation alone; label and timer remain available.
- Respect `prefers-reduced-motion`.
- User may hide the companion without losing timer control.
- Minimum target size 32×32 px; preferred 40×40 px.
- Do not rely on red/green alone for completion state.
- Audio cues are optional and independently mutable.
- Pixel art is decorative when adjacent text already communicates state.
