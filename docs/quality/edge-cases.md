# Edge Cases Register

- System clock moves backward or forward.
- Device sleeps for longer than planned duration.
- Main window closes while companion remains.
- Companion webview reloads.
- Process is killed during a pause.
- Completion command fires simultaneously from two windows.
- Stored active session has an open pause interval.
- Secondary monitor is unplugged.
- DPI scale changes.
- Click-through is enabled and shortcut registration fails.
- Notification permission denied.
- Database migration fails halfway.
- Update is attempted during active session.
- Custom duration is zero, negative or over four hours.
- Mission content contains emoji or non-Latin scripts.
