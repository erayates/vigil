# Production Roman Pixel UI Kit Brief — `PROD-UI-001`

## Dependency state

`request-user`. Agents must use the asset request protocol and stop before production-art integration.

## Required source groups

### Frames

- outer application frame,
- parchment panel 9-slice,
- stone panel 9-slice,
- dark panel 9-slice,
- active red tab/ribbon 9-slice.

### Controls

For green, red, orange and purple actions:

- default,
- hover,
- pressed,
- disabled,
- keyboard-focus treatment.

### Icons/ornaments

- focus, short break, long break,
- settings, statistics,
- priority high/medium/low,
- sessions, formation, focus time, total time,
- laurel, eagle, standard/banner, divider and corner ornaments.

## Rules

- No functional text baked into art.
- Transparent backgrounds where appropriate.
- Corners must survive 9-slice scaling.
- Source and export dimensions documented.
- Pixel palette remains consistent with the character set.
- Rights/provenance must be recorded.
